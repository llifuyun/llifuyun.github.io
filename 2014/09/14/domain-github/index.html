GitHub域名绑定

　　Github中绑定域名的方法很简单，具体步骤为：

  1. 在根目录中新建CNAME的文件，然后里面写入你需要绑定的域名，当然可以支持同时绑定多个域名，你只需要分多行编写即可；
  2. 将你的域名按CNAME的解析方式解析到你的GitHub二级域名即可。
